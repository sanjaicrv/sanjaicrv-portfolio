# Sanjai CRV — Portfolio

A modern, high-performance personal portfolio website showcasing my projects, skills, and achievements as a Backend-Focused Full Stack Developer.

🔗 **Live:** [sanjai-crv-portfolio](https://sanjaicrv.github.io/) _(update with your deployed URL)_

---

## 🚀 Tech Stack

| Layer        | Technologies                                      |
| ------------ | ------------------------------------------------- |
| **Frontend** | React.js, TypeScript, Vite                        |
| **Styling**  | Tailwind CSS, Framer Motion                       |
| **3D/VFX**   | Three.js (particle background), custom animations |
| **Icons**    | Lucide React, Devicon CDN                         |
| **Tooling**  | ESLint, PostCSS                                   |

---

## ✨ Features

- **Hero Section** — Animated entrance with typing text, orbiting tech icons, 3D profile card, and spinning CTA
- **About Section** — Fixed-scroll reveal with Education, Achievements, and Focus areas in a two-column layout
- **Selected Works** — Scroll-stacking project cards with star-border animations and 3D void zoom transition
- **Skills** — Interactive marquee hover system with categorized tech stack icons
- **Contact** — Clean contact form with social links
- **Footer** — Parallax reveal footer with smooth scroll integration
- **Responsive** — Fully responsive across desktop, tablet, and mobile
- **Cursor Effects** — Custom cursor follower and splash cursor (desktop)
- **Noise Texture** — Film grain overlay for a premium aesthetic

## 🛠️ Getting Started

### Prerequisites

- Node.js 18+
- npm or bun

### Install & Run

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
